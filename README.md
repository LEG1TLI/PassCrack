Hi! welcome to PassCrack. This tool is meant to be a password bruteforce tool using C++. I have no doubt there are many just like it, but this one is mine! This is v1.0.0, but this project will be consistently updated as I find time!

Features:
-4, 6, and 8 digit password bruteforcing (randomly generates numbers until a match is found)
-dictionary attack using a custom wordlist 

Coming features:
-ability to use output to attempt bruteforce on devices via USB port 
-use of custom wordlists (seclists, etc.)
-plain old password generator
